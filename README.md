# 2D Graphics Editor in C

A simple console-based graphics editor using a 2D character array.

## Features
- Draw Rectangle
- Draw Line
- Draw Triangle
- Draw Circle
- Delete Shapes
- Display Picture

## Technologies
- C Programming
- Arrays
- Structures
- Functions

## How to Run

```bash
gcc graphic_editor.c -o graphic_editor
./graphic_editor
```
